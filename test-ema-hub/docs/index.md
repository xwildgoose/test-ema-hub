# EMA Knowledge Hub

This site is the single source of content for the **EMA chatbot**. FastBots crawls it
on a schedule, so anything published here becomes part of the bot's knowledge.

- **Announcements** live on the [Clarion](clarion.md) page.
- **Documents** (policies, guides, forms) appear under *Documents* in the navigation.

To update the bot, edit or add a page here. FastBots picks up the change on its next
scheduled crawl -- no re-uploading, no clicking retrain.
